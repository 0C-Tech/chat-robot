Chat robot based on ChatGPT.

## Development setup

```bash
# Installation
$ npm i

# Development
$ npm run start
$ npm run build
$ npm run build:compress
$ npm run build:prod
$ npm run watch

# Code scaffolding
$ ng g component|directive|pipe|service|class|guard|interface|enum|module name

# Lint & test
$ npm run lint
$ npm run test

# Deploy
$ npm run deploy
```
